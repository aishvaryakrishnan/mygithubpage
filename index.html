Summer Internship May'15 - Aug'15

In the Summer of 2015 I worked as Software Engineering Intern with ServiceNow, Inc. to build a web application to manage customer data. I developed an end-to-end application leveraging the ServiceNow platform that enabled efficient management of customer data using automation. A customer clone is a dataset provided by a customer that mimics their data in a particular domain. This application is aimed at making the process of managing the customer clones and handling customer clone requests seamless as compared to the existing static implementation. Tools used: ServiceNow platform, Javascript, Glide.
